Tic Tac Toe - React

A simple Tic Tac Toe game built with React. This project demonstrates the basics of React state management, component structure, and user interaction.

Features:
1:Play classic Tic Tac Toe (X vs O) in your browser
2:Interactive UI with instant feedback
3:Displays the current player's turn
4:Announces the winner when the game ends
5:Prevents moves on filled squares or after the game is won

Getting Started:
  Prerequisites
  1:Node.js (v14 or higher recommended)
  2:npm or yarn
  
Installation:
1:Clone the respository: git clone https://github.com/moizkaseer/tic-tac-toe-react.git
2:Navigate to the project directory : cd tic-tac-toe-react/my-react-app
3:Install dependencies: npm install or yarn install

Running the Game 
Start the development server:
npm run dev or yarn run dev

How to Play
Click on any empty square to place your mark (X or O).
Players alternate turns.
The game announces the winner when three marks align horizontally, vertically, or diagonally.
No further moves are allowed after the game ends.
Customization
Feel free to modify the styles in App.css or extend the game with features like score tracking or a reset button!
